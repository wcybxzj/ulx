第一天 5.作业
4台机器3个网段通讯

=======================================================
网段:
default (NAT):192.168.122.0/24
ten (isolated network):10.10.10.0/24
two (isolated network):20.20.20.0/24
=======================================================
node1:
ip a:
eth0 192.168.122.101/24

ip r:
192.168.122.0/24  dev eth0 192.168.122.101
default via 192.168.122.102 dev eth0
=======================================================
node2:
开启ip_forward

ip a:
eth0 192.168.122.102/24
eth1 10.10.10.1/24

ip r:
10.10.10.0/24 dev eth1 10.10.10.1
192.168.122.0/24 dev eth0 192.168.122.102
default via 10.10.10.2 dev eth1
=======================================================
node3:
开启ip_forward

ip a:
eth1 10.10.10.2/24
eth3 20.20.20.2/24

ip r:
20.20.20.0/24 dev eth3 20.20.20.2
10.10.10.0/24 dev eth1 10.10.10.2
default via 10.10.10.1 dev eth1
=======================================================
node4:

ip a:
eth2 20.20.20.1/24

ip r:
20.20.20.0/24 dev eth2 20.20.20.1
default via 20.20.20.2 dev eth2
=======================================================

在node1 ping 20.20.20.2
