0.注意这里内网用的是vmware nat的192.168.91.0/24
 在windows不用vpn 也可以直接ping通,所以我的笔记和配置都改成1.0.0.0/8

1.内网:192.168.91.0/24  外网:2.0.0.0/8
========================================================================
2.拓扑
client:   vpnserver:     web:
2.2.2.2   2.2.2.1        192.168.91.3
          192.168.91.11
========================================================================
3.vpnserver:
开启ip_forward
ip a:
eth0:192.168.91.11/24
eth1:2.2.2.1/8

ip r:
10.8.0.0/24 dev tap0  proto kernel  scope link  src 10.8.0.1 
192.168.91.0/24 dev eth0  proto kernel  scope link  src 192.168.91.11 
2.0.0.0/8 dev eth1  proto kernel  scope link  src 2.2.2.1 

========================================================================
4.client:
ip a:
eth3:2.2.2.2/8

ip r:
10.8.0.0/24 dev tap0  proto kernel  scope link  src 10.8.0.2 
192.168.91.0/24 via 10.8.0.1 dev tap0 
2.0.0.0/8 dev eth3  proto kernel  scope link  src 2.2.2.2 
========================================================================
5.web:
ip a:
eth0:192.168.91.3/24

ip r:
192.168.91.0/24 dev eth0  proto kernel  scope link  src 192.168.91.3  metric 1 
default via 192.168.91.11 dev eth0  proto static
========================================================================
6.测试
